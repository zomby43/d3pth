Game Tips and Commands

Welcome to d3pth - simple ASCII Roguelike Game by z0mb0t

Controls:

  • Movement: Use the arrow keys, WASD, or on-screen buttons.

Commands:

  • drop [slot number]: Drops the item in the specified inventory slot.
  • use item [slot number]: Uses the item in the specified inventory slot. (example: some items can be used like "use item 2 up" (or down, left and right))
  • disable trap or attack trap: Disables a trap when standing on it.
  • upgrade [attribute]: Spend a skill point to upgrade an attribute (health, attack, defense).

For now, you can unrealistically stack items like swords and shields for more attributes! :)

Objective:

Explore the dungeon, defeat Enemies by moving in their direction, collect items, avoid or disable traps, and reach the eXit to progress to the next level.

Good Luck!
